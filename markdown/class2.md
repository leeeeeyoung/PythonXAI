## 🖥️ 一、用 Streamlit 做互動網頁（class2-1）

### 1️⃣ 輸入數字

```python
st.number_input("請輸入一個數字")
```

👉 會出現一個「可以加減的數字框」
👉 使用者可以用滑鼠或鍵盤輸入數字

📌 可以設定：

- 最小值、最大值
- 預設值
- 一次加多少

---

### 2️⃣ 判斷成績等第（如果…就…）

```python
if score >= 90:
    grade = "A"
```

📌 規則很像考試成績：

- 90 分以上 → A
- 80 分以上 → B
- 70 分以上 → C
- 60 分以上 → D
- 不及格 → F

👉 電腦會 **從上面一路檢查到下面**

---

### 3️⃣ 按鈕特效 🎈 ❄️

```python
st.button("點我")
```

- 點按鈕 → 放氣球 🎈
- 點按鈕 → 下雪 ❄️

📌 按鈕只有「被點到」那一瞬間才會做事

---

## 🔁 二、for 迴圈（重複做事情）（class2-2）

### 1️⃣ 重複說「hello」

```python
for i in range(5):
    print("hello")
```

👉 會印出 5 次 hello

---

### 2️⃣ 印出數字

```python
range(5)        # 0 到 4
range(2, 6)     # 2 到 5
range(2, 10, 2) # 2, 4, 6, 8
```

📌 `range` 就像：

> 「請電腦照這個數字規則，一個一個來」

---

## 🔺 三、數字金字塔 & 星星（class2-3）

### 1️⃣ 數字金字塔

```
1
22
333
4444
```

📌 做法：

- 第 1 行 → 1 印 1 次
- 第 2 行 → 2 印 2 次
- 以此類推

---

### 2️⃣ 箭頭金字塔 ⭐

- 用「空白」+「星星」
- 讓圖形看起來在中間

📌 `*` 可以重複很多次
📌 `\n` 代表「換一行」

## 📦 四、串列（list）＝裝東西的盒子（class2-4）

### 1️⃣ 各種串列

```python
[1, 2, 3]                 # 數字
["apple", "banana"]       # 文字
[1, "apple", True]        # 混合
[1, 2, [3, 4]]            # 裡面還有盒子
```

---

### 2️⃣ 取出盒子裡的東西

```python
L5[0]      # 第一個
L5[3][0]   # 盒子裡的盒子
```

📌 電腦數數字是 **從 0 開始**

---

### 3️⃣ 切片（挑一部分）

```python
L6[1:4]    # 從第 1 個到第 3 個
L6[::2]    # 每隔一個拿一次
```

👉 就像「挑玩具的一部分」

---

## 📏 五、串列的長度與操作（class2-5）

### 1️⃣ len()：有幾個東西

```python
len([1, 2, 3])  # 3
```

📌 **外層有幾個算幾個**
裡面的小盒子算 1 個

---

### 2️⃣ 看串列裡的每個東西

```python
for element in L1:
    print(element)
```

👉 電腦會一個一個拿出來看

---

### 3️⃣ 改串列

```python
append()  # 加東西
remove()  # 移除指定的
pop()     # 拿掉一個
sort()    # 排順序
```

📌 注意：

```python
b = a
```

👉 b 和 a **其實是同一個盒子**

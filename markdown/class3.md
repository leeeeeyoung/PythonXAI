## 📌 一、Streamlit 點餐機（class3-1）

### 🍔 這個程式在做什麼？

做一台 **點餐機**：

- 可以輸入餐點
- 按「加入」放進購物籃
- 可以刪掉不要的餐點

---

### 🧠 重要觀念

#### 1️⃣ `st.session_state`（記憶盒子）

- 就像 **電腦的記憶本**
- 可以記住你點過哪些餐點
- 不會因為畫面重整就消失

```python
ss = st.session_state
```

---

#### 2️⃣ 清單 `orders`

- 用來存「所有點的餐點」
- 像這樣：`["漢堡", "薯條", "可樂"]`

```python
if "orders" not in ss:
    ss.orders = []
```

👉 如果還沒有購物籃，就先準備一個空的

---

#### 3️⃣ 按鈕＋重新整理畫面

```python
ss.orders.append(foodInput)
st.rerun()
```

- 加入或刪除後
- 重新整理畫面，讓結果馬上更新

---

### ⭐ 小重點

- `session_state`：記住資料
- `list.append()`：加東西
- `list.pop()`：刪東西

## 📌 二、算數指定運算子（class3-2）

### ➕ 快速寫法（懶人版）

| 寫法      | 意思      |
| --------- | --------- |
| `a += b`  | a = a + b |
| `a -= b`  | a = a - b |
| `a *= b`  | a = a × b |
| `a /= b`  | a = a ÷ b |
| `a //= b` | 整數除法  |
| `a %= b`  | 取餘數    |
| `a **= b` | 次方      |

---

### 🧠 例子

```python
a = 10
a += 3
```

👉 a 會變成 **13**

---

## 📌 三、while 與 for 迴圈 + break

### 🔁 while 迴圈

👉 **只要條件成立就一直做**

```python
i = 0
while i < 5:
    print(i)
    i += 1
```

---

### 🛑 break（中途停止）

```python
if i == 3:
    break
```

👉 碰到 3 就直接停下來！

---

### 🔁 for 迴圈

```python
for i in range(1, 6):
    print(i)
```

👉 會跑 **1～5**

---

### ⭐ 小重點

- `while`：看條件
- `for`：跑固定次數
- `break`：強制停下

## 📌 六、字典 Dictionary（class3-6）

### 📦 什麼是字典？

像「姓名簿」：

- 左邊是 **名字（key）**
- 右邊是 **內容（value）**

```python
scores = {"國文": 90, "英文": 80}
```

---

### 🔍 常用操作

#### 取值

```python
scores["國文"]
```

#### 新增 / 修改

```python
scores["數學"] = 85
scores["國文"] = 95
```

#### 刪除

```python
scores.pop("英文")
```

---

### 🔁 走訪字典

| 指令       | 功能       |
| ---------- | ---------- |
| `keys()`   | 所有科目   |
| `values()` | 所有分數   |
| `items()`  | 科目＋分數 |

---

### 📊 成績平均

- 用字典存每科 10 筆成績
- 用迴圈算平均
